Points to remember

  • Images are created using <img> Tag. It is self enclosing
  • src attribute is used to set the location of the image
  • Recommended to create a folder with all the images and use relative path to set the images
  • Use alt attribute to set alternate text if the image is not able to load
  • U can use other attributes like width and height to set the height/width of the images
